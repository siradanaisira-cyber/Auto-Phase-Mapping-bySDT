Tech Stack:
HTML5 โครงสร้างหน้าเว็บแบบ Single Page Application ทำงานได้ทันทีบน GitHub Pages โดยไม่ต้องมี build step
Tailwind CSS Utility-first CSS framework จัดธีมสดใส โทนน้ำเงิน · เขียวอ่อน · ส้มอ่อน
Vanilla JS (ES6+) Rule-Based Engine สำหรับวิเคราะห์ Description → Section Code และรันเลข Running Number อัตโนมัติ
SheetJS อ่าน/เขียนไฟล์ Excel (.xlsx) และ CSV ทั้งฝั่งนำเข้าและส่งออก
PDF.js แกะ Text Layer จากไฟล์ PDF เพื่อดึงคอลัมน์ Item No. และ Description
Supabase PostgreSQL Backend บันทึกผลลัพธ์ผ่าน supabase.from('phase_mappings').insert()
