Vanilla JS (ES6+) Rule-Based Engine สำหรับวิเคราะห์ Description → Section Code และรันเลข Running Number อัตโนมัติ

PDF.js แกะ Text Layer จากไฟล์ PDF เพื่อดึงคอลัมน์ Item No. และ Description
